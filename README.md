# Проверка свойства древочисленности (ордерево)

## Описание

Программа проверяет, является ли заданный орграф ордеревом по определению, и проверяет свойство древочисленности.

## Сборка проекта

```bash
mkdir build
cd build
cmake ..
make
```

## Запуск программы

### Автоматический режим (все тесты)
```bash
./ordtree_checker
```
Программа автоматически прогонит все тесты из папки `tests/` и сохранит результаты в `outputs/`.

### Ручной режим (один файл)
```bash
./ordtree_checker <входной_файл> <выходной_файл>
```

Пример:
```bash
./ordtree_checker tests/test1_ordtree.txt my_output.txt
```

## Формат входного файла

Первая строка: `n m` — количество вершин и рёбер
Следующие `m` строк: `u v` — ребро из вершины u в вершину v

Вершины нумеруются от 1 до n.

Пример:
```
5 4
1 2
1 3
2 4
2 5
```

## Формат выходного файла

Программа выводит подробный отчёт с анализом всех условий:

1. **Проверка наличия корня** - указываются полустепени захода всех вершин
2. **Проверка полустепеней захода** - указываются вершины с неправильными полустепенями
3. **Проверка достижимости** - перечисляются недостижимые вершины
4. **Проверка древочисленности** - проверка условия q(G) = p(G) - 1
5. **Финальное заключение** - итоговый вывод о свойствах графа
